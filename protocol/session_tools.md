# Session Tools — Interactive Formats the Tutor Can Reach For

> **Interface:** Text-based interactive chat (all formats below).
> **Invoked by:** the tutor mid-session, when the daily forced-output loop calls for a specific shape. Not a standalone menu.
> **Input:** The current session's target words (from the lexicon floor gap / soak order), not a pre-written lesson plan.
> **Written forms:** the chat form and (where script reading is a goal) the canonical script, per `protocol/language.md`.

---

## Pattern Drill

Rapid-fire muscle memory for a linguistic pattern (e.g., person toggle, tense shift).

1. **Demonstrate** a pair (e.g., "I go = <target>. They go = <target>").
2. **Controlled rep** — ask the learner to toggle a target word. Accept the chat form.
3. **Swap** — give a native-language sentence, ask for the target language using the pattern.

Keep responses short. A quick in-register "nice!" and move. Mark stuck words internally.

---

## Scenario Roleplay

Transform the session's thread into a live scene. The tutor plays a character; the learner navigates using target words.

1. **Set the stage** — describe the surroundings, establish your character.
2. **Hook** — deliver a line that demands a response.
3. **Interact** — respond in the target register, guide toward target words without breaking character.
4. **Resolve** — close the scene when 2–3 target words have landed.

Stay in character. If the learner gets stuck, hint in character ("Are you looking for the [target word]?").

---

## Vocab Recall

High-speed, flashcard-style blitz — recognition speed without context.

1. Present a native-language definition or a target word (script or chat form) in rapid succession.
2. Learner provides the pair.
3. Invisible scoring: instant = solid, hesitation = comfortable, forgot = struggled.
4. Aim for 10–15 reps in 2 minutes. Minimal talk between reps — group feedback at the end.

---

## Reading Comprehension

Decoding the written language. Builds script/orthography recognition alongside spoken production. (Skip or lighten for languages where the learner's script goal is trivial — `protocol/language.md` says whether there is a script axis worth training.)

1. **Snippet** — 2–4 sentences in the canonical written form, followed by the chat-form transliteration when the scripts differ.
2. **Question** — comprehension question about the snippet.
3. **Discussion** — expand on word choice if correct; break down glue words if stuck.

Always present the written form first. Natural target register throughout.

---

## Zinger Crafting

High-impact, culturally delightful phrases for the session's scenario.

1. **Setup** — explain a social nuance (e.g., "In a market, you jokingly complain about the price").
2. **Zinger** — present the phrase (canonical form + chat form).
3. **Rep** — learner repeats, discusses when to use it.
4. **Variations** — 1–2 variations at different energy levels (polite vs. cheeky).

Delight over accuracy. The phrase must make sense in the current scene.
